\begin{abstract}
Federated learning under concept drift faces a basic design choice: aggregate all client models globally, or maintain separate models per concept group?
We provide a sharp answer in a canonical Gaussian linear model:
concept-level aggregation outperforms global aggregation if and only if the per-concept signal-to-noise ratio $\SNR = Kn B_j^2 / (\sigma^2 d) > C{-}1$.
Four relaxations---unbalanced proportions, noisy concept labels, high-dimensional proportional regime, and anisotropic features---yield a unified per-concept decision rule (Theorem~\ref{thm:unified}).
An empirical-Bayes shrinkage estimator interpolates adaptively between the two strategies, outperforming the better pure strategy in 72\% of 108 synthetic configurations.
The noisy-label extension reveals a surprising mechanism: \emph{imperfect clustering acts as implicit shrinkage}, biasing concept-level estimates toward the global mean.
This explains why CFL~\citep{sattler2021clustered}, whose gradient clustering has non-zero error rate, matches or exceeds oracle concept-level aggregation in practice when per-concept sample sizes are small ($K/C \approx 1.5$): the theory's bias penalty is negligible relative to the shared variance, while additional finite-sample regularization makes imperfect clustering genuinely beneficial beyond the linear model.
We validate this explanation through three tests: (i)~controlled noise injection matches the predicted threshold inflation, (ii)~K-scaling experiments confirm the CFL--Oracle gap shrinks monotonically as $K/C$ grows (from $+17$pp at $K{=}4$ to $+3.7$pp at $K{=}20$, 5~seeds each), and (iii)~SCAFFOLD, which reduces gradient variance without shrinkage, fails catastrophically under concept drift.
Across 23 methods on CIFAR-100 frozen features (5~seeds), theory-calibrated concept tracking matches Oracle ($p{=}0.84$) and significantly outperforms FedAvg ($+3.4$pp, $p{=}0.002$).
\end{abstract}

\section{Introduction}
\label{sec:intro}
% =============================================================================

Federated learning~(FL) enables multiple clients to collaboratively train models without sharing raw data \citep{mcmahan2017communication}. In many practical deployments---autonomous vehicles, mobile health, IoT---the data distribution shifts over time due to \emph{concept drift}: clients experience different tasks or environments at different times \citep{jothimurugesan2023federated, panchal2023flash}. We refer to these latent task types as \emph{concepts}.

A fundamental question arises: \emph{should the server aggregate all client models globally, or maintain separate models per concept group?} Global aggregation uses all $K$ clients, reducing variance but introducing interference bias. Concept-level aggregation eliminates bias but reduces the effective sample size to $K/C$ per group. Clustered FL methods (IFCA~\citep{ghosh2020efficient}, CFL~\citep{sattler2021clustered}) focus on \emph{how} to cluster, assuming it helps. None characterize \emph{when} concept-level aggregation is worth the variance cost.

\textbf{Contributions.}
\begin{enumerate}[leftmargin=*,topsep=2pt,itemsep=1pt]
    \item \textbf{Sharp crossover and unified extensions} (\S\ref{sec:theory}--\ref{sec:extensions}): Under balanced concepts, concept-level aggregation outperforms global iff $\SNR > C - 1$. Four relaxations---unbalanced proportions ($\to (1{-}\pi_j)/\pi_j$), noisy labels ($\to (C{-}1)/(1{-}\rho^2)$), proportional regime ($d/n \to \gamma$), anisotropic features ($d \to r_{\mathrm{eff}}$)---yield a unified per-concept decision rule (Theorem~\ref{thm:unified}). Minimax lower bounds confirm both cost floors.
    \item \textbf{Imperfect clustering as implicit shrinkage} (Corollary~\ref{cor:implicit}): Any clustering method with symmetric error rate $\eta$ produces an estimator with implicit shrinkage $\lambda_{\mathrm{imp}} = \eta C/(C{-}1)$ toward the global mean. This explains CFL's strong performance when $K/C$ is small: the bias penalty is negligible relative to the shared variance, and in practice the implicit regularization compensates for additional finite-sample effects. The theory predicts CFL's advantage diminishes as $K$ grows---confirmed by K-scaling experiments.
    \item \textbf{Adaptive shrinkage estimator} (Proposition~\ref{thm:shrinkage}): An empirical-Bayes estimator interpolates between global and concept-level, outperforming the better pure strategy in 72\% of 108 configurations. A per-concept variant adapts the shrinkage rate to each concept's group size and separation.
    \item \textbf{Comprehensive validation}: 100\% crossover alignment across 609 synthetic configs (5~seeds each), spanning the base case, unbalanced proportions, noisy labels, proportional regime, and anisotropic features. On CIFAR-100 (23~methods, 5~seeds), calibrated concept tracking matches Oracle ($p{=}0.84$), significantly beats FedAvg ($+3.4$pp, $p{=}0.002$), and the CFL--Oracle gap shrinks monotonically with $K$ as predicted. Controlled noise injection and SCAFFOLD's catastrophic failure provide further causal evidence.
\end{enumerate}

\section{Problem Setting}
\label{sec:setting}
% =============================================================================

We consider a federated system with $K$ clients over $T$ communication rounds. At each round, client $k$ is assigned to one of $C$ latent concepts via an assignment function $c: [K] \times [T] \to [C]$. Let $\cC_j(t) := \{k : c(k,t) = j\}$ denote the set of clients assigned to concept $j$ at time $t$.

\textbf{Data model.} For concept $j$, each client $k \in \cC_j(t)$ draws $n$ i.i.d.\ samples:
$x \sim \mathcal{N}(0, I_d)$, $y = \langle \wstar_j, x \rangle + \epsilon$, $\epsilon \sim \mathcal{N}(0, \sigma^2)$.
We work with centered isotropic features; when the original design has known shared $\Sigma$, one pre-whitens via $\tilde{x} = \Sigma^{-1/2}(x - \mu)$.

\textbf{Concept separation.} We define the bias of concept $j$ relative to the concept centroid $\wbar^* := \frac{1}{C}\sum_j \wstar_j$:
\begin{equation}
    B_j^2 := \|\wstar_j - \wbar^*\|^2.
\end{equation}

\textbf{Aggregation strategies.} \emph{Global OLS}: pool all $Kn$ samples, compute $\what_{\mathrm{glob}}$. \emph{Concept-level OLS}: pool only the $(K/C)n$ samples from $\cC_j$, compute $\what_j$. The excess risk is $\cE_j(\what) := \|\what - \wstar_j\|^2$.

\textbf{Assumptions.} (A1)~balanced concepts ($|\cC_j| = K/C$); (A2)~isotropic Gaussian design; (A3)~$n \ge \Omega(d)$.

\section{Theory}
\label{sec:theory}
% =============================================================================

\subsection{Bias-Variance Decomposition}

\begin{proposition}[Bias-Variance Decomposition]
\label{prop:decomposition}
Under (A1)--(A3) with $\Sigma = I_d$, the expected excess risks satisfy:
\begin{align}
    \E[\cE_j(\what_{\mathrm{glob}})] &= B_j^2 + \frac{\sigma^2 d}{Kn}(1 + o(1)), \label{eq:global_risk} \\
    \E[\cE_j(\what_j)] &= \frac{\sigma^2 d}{(K/C)n}(1 + o(1)), \label{eq:concept_risk}
\end{align}
where $o(1) = O(d / \min(Kn, (K/C)n))$.
\end{proposition}

Global pays interference bias $B_j^2$ but has lower variance ($1/(Kn)$ vs $C/(Kn)$).

\subsection{Crossover Condition}

\begin{theorem}[Crossover Condition]
\label{thm:crossover}
Under (A1)--(A3), as $n/d \to \infty$, the concept-level estimator has lower expected excess risk than the global estimator for concept $j$ if and only if
\begin{equation}
    \SNR := \frac{Kn \cdot B_j^2}{\sigma^2 d} > C - 1.
    \label{eq:crossover}
\end{equation}
\end{theorem}

\emph{Proof.} Setting global risk $>$ concept-level risk: $B_j^2 > \sigma^2 d(C{-}1)/(Kn)$; rearranging yields~\eqref{eq:crossover}. \qed

\textbf{Relation to prior theory.} Prior clustered FL theory~\citep{ghosh2020efficient, guo2024fedrc} analyzes \emph{convergence and cluster recovery}---given that one wants to cluster, how fast can the algorithm find the right assignments? \citet{chen2023minimax} establish a minimax threshold between FedAvg and local training (the global-vs-local binary). Our work addresses the logically prior question of \emph{when} to cluster, and provides: (i)~an exact threshold for $C$-way aggregation with four relaxations; (ii)~a formal explanation of why imperfect clustering can beat oracle aggregation (Corollary~\ref{cor:implicit}); and (iii)~testable predictions validated on 23 methods.

\subsection{Minimax Lower Bounds}

\begin{theorem}[Minimax Lower Bounds]
\label{thm:lower}
Under (A1)--(A2) with $\frac{1}{C}\sum_j \|\wstar_j - \wbar^*\|^2 \ge B^2$: (i)~any single estimator incurs average risk $\ge B^2$; (ii)~any concept-level estimator incurs $\E[\|\what_j - \wstar_j\|^2] \ge \Omega(\sigma^2 d / ((K/C)n))$.
\end{theorem}

The global bottleneck exceeds the concept-level bottleneck when $KnB^2/(\sigma^2 d) > C$, matching~\eqref{eq:crossover} up to $C/(C{-}1) \to 1$. Proofs in Appendix~\ref{app:lower}.

\begin{remark}[Non-Stationary Extension]
Under piecewise-stationary drift with stability period $\tau$, a client $s$ rounds post-switch has effective samples $n_{\mathrm{eff}}(s) = n \cdot \min(s, W)$. This implies a transient period $s^* \approx \sigma^2 d(C{-}1)/(KnB_j^2)$ after concept switches during which global aggregation may be preferable. This is heuristic: the formal extension requires additional regularity conditions. Regime boundary experiments (Appendix~\ref{app:regime}) validate the qualitative prediction.
\end{remark}

\section{Adaptive Shrinkage Estimator}
\label{sec:shrinkage}
% =============================================================================

A principled interpolation arises from empirical Bayes / James-Stein shrinkage~\citep{james1961estimation,efron1973stein}. Treating concept-level estimates as noisy observations $\what_j \approx \mathcal{N}(\wstar_j, \frac{\sigma^2}{(K/C)n} I_d)$ with empirical prior $\wstar_j \sim \mathcal{N}(\wbar^*, \sigma_B^2 I_d)$:

\begin{proposition}[Empirical-Bayes Shrinkage]
\label{thm:shrinkage}
Under the Gaussian approximation $\what_j \sim \mathcal{N}\!\bigl(\wstar_j,\, \tfrac{\sigma^2}{(K/C)n} I_d\bigr)$, the posterior mean is:
\begin{equation}
    \what_j^{\mathrm{shrink}} = (1 - \lambda)\what_j + \lambda \bar{\what},
    \label{eq:shrinkage}
\end{equation}
where $\bar{\what} = \frac{1}{C}\sum_j \what_j$ and the shrinkage coefficient is
\begin{equation}
    \lambda = \frac{\sigma^2 / ((K/C)n)}{\sigma^2 / ((K/C)n) + \sigma_B^2}.
    \label{eq:lambda}
\end{equation}
\end{proposition}

\textbf{Practical estimation.} $\hat{\sigma}_B^2 = \max\!\bigl(\frac{1}{(C-1)d}\sum_{j} \|\what_j - \bar{\what}\|^2 - \frac{\sigma^2}{(K/C)n},\; 0\bigr)$ makes $\lambda$ data-adaptive with no tuning. APFL~\citep{deng2020adaptive} and Ditto~\citep{li2021ditto} are special cases with fixed $\lambda$.

\section{Extensions Beyond the Base Case}
\label{sec:extensions}
% =============================================================================

The base crossover relies on four assumptions: balanced concepts, oracle labels, isotropic features, $d \ll Kn$. We relax each, yielding modified thresholds that combine into a single unified condition. All proofs are in Appendix~\ref{app:extensions}.

\textbf{(I) Unbalanced proportions.} Let concept $j$ contain fraction $\pi_j$ of clients. The bias becomes $B_{j,\pi}^2 = \|\wstar_j - \sum_\ell \pi_\ell \wstar_\ell\|^2$.

\begin{theorem}[Unbalanced Crossover]
\label{thm:unbalanced}
Concept-level aggregation wins for concept $j$ iff $Kn B_{j,\pi}^2/(\sigma^2 d) > (1{-}\pi_j)/\pi_j$.
Rare concepts ($\pi_j \ll 1$) face threshold $\approx 1/\pi_j$: fewer clients means higher variance.
\end{theorem}

\textbf{(II) Noisy concept labels.} Assign each client with symmetric noise rate $\eta$; let $\rho = \eta C/(C{-}1)$.

\begin{theorem}[Noisy-Label Crossover]
\label{thm:noisy}
Under noise $\eta$: $\SNR > (C{-}1)/(1{-}\rho^2)$. When $\eta \ge (C{-}1)/C$, concept-level is dominated.
\end{theorem}

The maximum tolerable noise is $\eta_{\max} = \frac{C-1}{C}\sqrt{1 - (C{-}1)/\SNR}$---a stopping criterion for concept-inference algorithms. Crucially, noisy clustering acts as \emph{implicit shrinkage}: the contaminated estimate converges to $(1{-}\rho)\wstar_j + \rho\wbar^*$ (Lemma~\ref{lem:contamination}, Appendix~\ref{app:extensions}).

\begin{corollary}[Imperfect Clustering as Implicit Shrinkage]
\label{cor:implicit}
Any clustering algorithm that assigns clients with symmetric error rate $\eta$ produces a concept-level estimator with implicit shrinkage coefficient $\lambda_{\mathrm{imp}} = \rho = \eta C/(C{-}1)$. Its excess risk is:
\begin{equation}
    \E[\cE_j] = \rho^2 B_j^2 + \frac{\sigma^2 Cd}{Kn}(1{+}o(1)).
    \label{eq:implicit_risk}
\end{equation}
Compared to the global estimator (risk $B_j^2 + \sigma^2 d/(Kn)$), the imperfect estimator beats global under a \emph{relaxed} crossover threshold $(1{-}\rho^2)B_j^2 > \sigma^2 d(C{-}1)/(Kn)$.
Compared to the oracle (risk $\sigma^2 Cd/(Kn)$), the imperfect estimator pays an additional bias $\rho^2 B_j^2$, but when $K/C$ is small this penalty is negligible relative to the shared variance.
As $K/C \to \infty$, the oracle variance vanishes and the bias $\rho^2 B_j^2$ becomes the dominant cost.
\end{corollary}

\begin{proposition}[Practical Oracle Variance Inflation]
\label{prop:variance_inflation}
Under $E$ epochs of SGD with step size $\alpha$ on a concept-level group of size $n_j = \pi_j Kn$, the practical oracle's generalization risk satisfies:
\begin{equation}
    R_{\mathrm{oracle}}^{\mathrm{SGD}}(n_j) = \frac{\sigma^2 d_{\mathrm{eff}}}{n_j} + \Delta_{\mathrm{ovf}}(E, \alpha, n_j, d_{\mathrm{eff}}),
    \label{eq:oracle_sgd}
\end{equation}
where $\Delta_{\mathrm{ovf}} \ge 0$ is the overfitting penalty, increasing in $E$ and decreasing in $n_j$. The imperfect estimator with contamination $\rho$ incurs:
\begin{equation}
    R_{\mathrm{imp}}^{\mathrm{SGD}} = \rho^2 B_j^2 + \frac{\sigma^2 d_{\mathrm{eff}}}{n_j} + \Delta_{\mathrm{ovf}} - \Delta_{\mathrm{reg}}(\rho),
    \label{eq:imp_sgd}
\end{equation}
where $\Delta_{\mathrm{reg}}(\rho) \ge 0$ captures the regularisation benefit of label contamination (analogous to label smoothing). The imperfect estimator beats the practical oracle when
\begin{equation}
    \rho^2 B_j^2 < \Delta_{\mathrm{reg}}(\rho).
    \label{eq:practical_crossover}
\end{equation}
\end{proposition}

\textbf{Proof sketch.} Decompose the generalization error into irreducible noise, approximation bias, and optimization excess. The oracle, trained on $n_j$ in-distribution samples, incurs optimization excess $\Delta_{\mathrm{ovf}}$ from fitting training noise. The imperfect estimator trains on the same $n_j$ samples but with fraction $\rho$ from other concepts. This contamination introduces bias $\rho^2 B_j^2$ (Lemma~\ref{lem:contamination}) but simultaneously prevents the optimizer from fitting concept-specific noise patterns, reducing the optimization excess by $\Delta_{\mathrm{reg}}$. The effect is analogous to label smoothing~\citep{szegedy2016rethinking}: replacing exact labels with a mixture toward the class mean reduces model confidence and overfitting. In the small-$n_j$ regime ($K/C \approx 1.5$), each group has 1--2 clients, making $\Delta_{\mathrm{ovf}}$ large; in the large-$K$ regime, $n_j \to \infty$ and $\Delta_{\mathrm{ovf}} \to 0$, recovering the OLS result where the oracle always dominates. Full derivation in Appendix~\ref{app:variance_inflation}.

\textbf{Testable predictions.} Proposition~\ref{prop:variance_inflation} yields three predictions: (i)~the CFL--Oracle gap should be positive when $K/C$ is small and shrink as $K$ grows (less overfitting with larger groups); (ii)~increasing $E$ should increase CFL's advantage (more overfitting for contamination to regularize); (iii)~methods with implicit shrinkage (CFL, FedProx, FLUX) should outperform methods without (SCAFFOLD). We verify all three in~\S\ref{sec:calibrated}.

\textbf{(III) Proportional regime} ($d/(Kn) \to \gamma_G > 0$).

\begin{theorem}[Proportional-Regime Crossover]
\label{thm:proportional}
$\SNR > (C{-}1)/[(1{-}C\gamma_G)(1{-}\gamma_G)]$. Near the interpolation boundary $C\gamma_G \to 1^-$, the threshold $\to \infty$.
\end{theorem}

\textbf{(IV) Anisotropic features.} With $x \sim \mathcal{N}(0, \Sigma)$, define $r_{\mathrm{eff}} = (\sum_i \lambda_i)^2/\sum_i \lambda_i^2$.

\begin{theorem}[Effective-Dimension Crossover]
\label{thm:aniso}
In the spiked model with concept separation in the signal subspace: $Kn B_{j,\Sigma}^2/(\sigma^2 r \cdot s) > C{-}1$, where $r$ is the signal rank and $s$ the signal eigenvalue.
\end{theorem}

Pre-trained features have $r_{\mathrm{eff}} \ll d$; the theorem predicts that $r_{\mathrm{eff}}$---not $d$---determines the crossover. We verify this in \S\ref{sec:cifar}.

\textbf{Unified crossover.} Combining all four:

\begin{theorem}[Unified Crossover]
\label{thm:unified}
With proportions $\boldsymbol{\pi}$, noise $\eta$ ($\rho < 1$), effective dimension $d_{\mathrm{eff}}$, aspect ratios $\gamma_j = d_{\mathrm{eff}}/(\pi_j Kn)$:
\begin{equation}
    \frac{\pi_j Kn \cdot B_{j,\pi,\Sigma}^2}{\sigma^2 d_{\mathrm{eff}}} > \frac{1-\pi_j}{\pi_j} \cdot \frac{1}{(1-\rho^2)(1-\gamma_j)(1-\pi_j\gamma_j)}.
    \label{eq:unified}
\end{equation}
Under base assumptions this reduces to $\SNR > C{-}1$. Of the four factors, only anisotropy \emph{lowers} the effective threshold; the others all raise it.
\end{theorem}

\subsection{CIFAR-100 Bridge: Validating the Anisotropic Extension}
\label{sec:cifar}

We now test the extended theory (\S\ref{sec:extensions}) on real data. CIFAR-100 with frozen ResNet-18 features ($d = 128$, $C=4$ concepts, $K = 10$ clients) provides a natural testbed: frozen features are highly anisotropic, so Theorem~\ref{thm:aniso} predicts that $r_{\mathrm{eff}}$---not $d$---should enter the SNR.

\textbf{Setup.} We fit one-vs-rest linear heads $W_j \in \R^{d \times n_{\mathrm{classes}}}$ per concept and measure $B_j^2 = \|W_j - \bar{W}\|_F^2$. Two SNR calculations: (i) the \emph{raw} SNR uses $d_{\mathrm{eff}} = d \times n_{\mathrm{classes}} = 2560$; (ii) the \emph{theory-corrected} SNR uses $d_{\mathrm{eff}}^{\mathrm{corr}} = r_{\mathrm{eff}} \times n_{\mathrm{classes}} \approx 430$, following Theorem~\ref{thm:aniso}'s prediction that effective rank replaces raw dimension.

\emph{High-SNR} (disjoint labels): $\SNR = 9.4 \pm 2.1 \gg C{-}1 = 3$ under either metric, Oracle wins empirically (83--88\% vs 66--75\%), 3/3 match.

\emph{Low-SNR} (shared labels): Raw $\SNR = 0.83 \pm 0.19 < C{-}1$, \emph{incorrectly} predicting FedAvg dominance. But the theory-corrected $\SNR^{\mathrm{corr}} = 4.9 \pm 1.1 > C{-}1$, \emph{correctly} predicting Oracle wins (59--63\% vs 44--54\%). Under the corrected dimension, all 6 seeds match.

\textbf{This is not post-hoc.} Theorem~\ref{thm:aniso} predicts that when concept separation lies in the signal subspace ($\|\Pi_r(\wstar_j - \wbar^*)\|/\|\wstar_j - \wbar^*\| > 0.95$, verified), the crossover uses $r_{\mathrm{eff}} \approx 22 \ll d = 128$ (Appendix~\ref{app:cifar_diag}), inflating effective SNR above threshold.

\subsection{Practical Transfer: Calibrated Concept Tracking}
\label{sec:calibrated}

We test the shrinkage principle on FedProTrack (FPT), a Bayesian concept-identity tracker. \section{Related Work}
\label{sec:related}
% =============================================================================

\textbf{Clustered FL.} IFCA~\citep{ghosh2020efficient}, CFL~\citep{sattler2021clustered}, and FedRC~\citep{guo2024fedrc} address \emph{how to cluster}; we address \emph{when to cluster}.

\textbf{Personalized FL.} APFL~\citep{deng2020adaptive}, Ditto~\citep{li2021ditto}, and pFedMe~\citep{t2020personalized} use fixed or heuristic mixing. Our shrinkage estimator derives $\hat\lambda$ from data, and Corollary~\ref{cor:implicit} shows imperfect clustering \emph{is} a form of shrinkage, unifying clustered and personalized FL.

\textbf{FL under concept drift.} FedDrift~\citep{jothimurugesan2023federated}, Flash~\citep{panchal2023flash}, FedCCFA~\citep{chen2024fedccfa}, and FedGWC~\citep{licciardi2025fedgwc} propose drift-aware algorithms without characterizing when concept-level treatment is worth the variance cost. SCAFFOLD~\citep{karimireddy2020scaffold} corrects client gradient drift but fails under temporal concept drift (\S\ref{sec:calibrated}).

\textbf{Theory.} \citet{chen2023minimax} establish a minimax threshold between FedAvg and local training (global vs.\ local). We extend to $C$-way concept aggregation with four relaxations and a data-adaptive estimator. \citet{fedem2024convergence} show heterogeneity can accelerate federated EM---complementary to our implicit-shrinkage finding. FedStein~\citep{fedstein2024} applies James-Stein to batch normalization; we use shrinkage to characterize the \emph{decision boundary} between strategies.

\section{Conclusion}
\label{sec:conclusion}
% =============================================================================

We characterized when concept-level aggregation helps under federated concept drift via a sharp crossover ($\SNR > C{-}1$) with four extensions unified in Theorem~\ref{thm:unified}. The key finding is that imperfect clustering acts as implicit shrinkage (Corollary~\ref{cor:implicit}), explaining why CFL and FedProx outperform oracle aggregation when $K/C$ is small---validated by noise injection, monotonic K-scaling, and SCAFFOLD's failure. This bridges clustered and personalized FL: these methods succeed not because they cluster well, but because they shrink appropriately.

\textbf{Limitations.} The Gaussian linear model does not extend to nonlinear settings or time-varying $C(t)$. The symmetric-noise model is a simplification---real clustering may merge concepts entirely. Experiments use frozen features; end-to-end training behavior is future work.

\section{Proof of Proposition~\ref{prop:decomposition}}
\label{app:prop1}

\textbf{Global estimator bias.} Under balanced concepts with $\Sigma = I_d$, pooling all $Kn$ samples mixes data from $C$ regression problems $y = \langle \wstar_j, x\rangle + \epsilon$. The population OLS solution on this mixture minimizes $\E[(y - \langle w, x\rangle)^2]$ over the mixture distribution $\frac{1}{C}\sum_j P_j$. Since $\E[xy] = \frac{1}{C}\sum_j \wstar_j = \wbar^*$ and $\E[xx^\top] = I_d$, the population minimizer is $\wbar^*$, not $\wstar_j$.

\textbf{Global estimator excess risk.} The excess risk for concept $j$ decomposes as:
$\E[\|\what_{\mathrm{glob}} - \wstar_j\|^2] = \|\wbar^* - \wstar_j\|^2 + \E[\|\what_{\mathrm{glob}} - \wbar^*\|^2] = B_j^2 + \E[\|\what_{\mathrm{glob}} - \wbar^*\|^2].$
The cross-term $2(\wbar^* - \wstar_j)^\top \E[\what_{\mathrm{glob}} - \wbar^*]$ vanishes since $\E[\what_{\mathrm{glob}}] = \wbar^*$. The variance term is $\E[\|\what_{\mathrm{glob}} - \wbar^*\|^2] = \sigma^2 \mathrm{tr}((X^\top X)^{-1})$. By random matrix theory, $\frac{1}{Kn}X^\top X \to I_d$ as $n/d \to \infty$, so $\mathrm{tr}((X^\top X)^{-1}) = d/(Kn)(1 + O(d/(Kn)))$.

\textbf{Concept-level estimator.} Using only the $(K/C)n$ samples from $\cC_j$, the OLS estimator $\what_j$ is unbiased for $\wstar_j$. By the same argument, $\E[\|\what_j - \wstar_j\|^2] = \sigma^2 d / ((K/C)n)(1 + O(d/((K/C)n)))$.

\section{Proof of Theorem~\ref{thm:lower}}
\label{app:lower}

\textbf{Part 1 (Global bottleneck):} For any single estimator $\what$ (possibly random), the average excess risk satisfies:
\begin{align*}
\frac{1}{C}\sum_{j=1}^C \|\what - \wstar_j\|^2 &= \frac{1}{C}\sum_j \|(\what - \wbar^*) + (\wbar^* - \wstar_j)\|^2 \\
&= \|\what - \wbar^*\|^2 + \frac{2}{C}\sum_j (\what - \wbar^*)^\top (\wbar^* - \wstar_j) + \frac{1}{C}\sum_j \|\wstar_j - \wbar^*\|^2.
\end{align*}
The cross-term vanishes because $\sum_j (\wbar^* - \wstar_j) = 0$ by the definition $\wbar^* = \frac{1}{C}\sum_j \wstar_j$. Hence
$\frac{1}{C}\sum_j \|\what - \wstar_j\|^2 = \|\what - \wbar^*\|^2 + \frac{1}{C}\sum_j \|\wstar_j - \wbar^*\|^2 \ge B^2.$
This is a deterministic inequality holding for \emph{any} $\what$---even the infeasible choice $\what = \wbar^*$---and establishes that no single estimator can avoid the average bias $B^2$.

\textbf{Part 2 (Concept-level bottleneck):} The concept-level estimator $\what_j$ uses $(K/C)n$ i.i.d.\ Gaussian samples with $\Sigma = I_d$. By the standard minimax lower bound for $d$-dimensional linear regression~\citep[Theorem~2.5]{tsybakov2009introduction}, $\inf_{\what_j} \sup_{\wstar_j} \E[\|\what_j - \wstar_j\|^2] \ge \sigma^2 d / (2(K/C)n)$.

\section{Baseline Implementation Details}
\label{app:baselines}

\textbf{IFCA.} Each client evaluates the MSE of all $C$ cluster heads on its local data, selects the best, trains locally, and uploads. The server averages uploads per cluster. This is a linear-Gaussian analogue of \citet{ghosh2020efficient}; the original IFCA uses gradient descent, which we replace with closed-form OLS.

\textbf{CFL.} After one round of global OLS, we compute pairwise cosine similarity of client gradient updates $\nabla_k = \what_k^{\mathrm{local}} - \what^{\mathrm{global}}$. Clients are clustered hierarchically with $C$ clusters. Each client is served by its cluster's averaged model. This is a linear-Gaussian analogue of \citet{sattler2021clustered}.

\textbf{APFL.} For each $\alpha \in \{0.2, 0.5, 0.8\}$, we compute $\what_k^{\mathrm{apfl}} = \alpha \what_k^{\mathrm{local}} + (1-\alpha)\what^{\mathrm{global}}$ and evaluate MSE. We report the best $\alpha$ per configuration. This is a tuned linear-Gaussian analogue of \citet{deng2020adaptive}.

\section{Proofs of Extensions (\S\ref{sec:extensions})}
\label{app:extensions}

\textbf{Proof of Theorem~\ref{thm:unbalanced}.}
Under unbalanced proportions, the global OLS population minimizer is $\wbar^*_\pi = \sum_\ell \pi_\ell \wstar_\ell$. The excess risk for concept $j$ decomposes as $\E[\cE_j(\what_{\mathrm{glob}})] = B_{j,\pi}^2 + \sigma^2 d/(Kn)(1{+}o(1))$ and $\E[\cE_j(\what_j)] = \sigma^2 d/(\pi_j Kn)(1{+}o(1))$. Setting the global risk larger: $B_{j,\pi}^2 > \sigma^2 d(1{-}\pi_j)/(\pi_j Kn)$, yielding the bound in Theorem~\ref{thm:unbalanced}. \qed

\textbf{Proof of Theorem~\ref{thm:noisy}.}
\begin{lemma}[Contamination Bias]
\label{lem:contamination}
Under label noise $\eta$ with balanced concepts, the concept-level OLS estimator using noisy group $\hat{C}_j$ converges to $\widetilde{w}_j = (1{-}\rho)\wstar_j + \rho\wbar^*$ where $\rho = \eta C/(C{-}1)$.
\end{lemma}
\begin{proof}
The fraction of data in $\hat{C}_j$ from concept $\ell$ is $q_{j\ell} = 1{-}\eta$ if $\ell{=}j$, and $\eta/(C{-}1)$ otherwise. The OLS target is $\widetilde{w}_j = \sum_\ell q_{j\ell} \wstar_\ell = (1{-}\eta)\wstar_j + \frac{\eta}{C-1}(C\wbar^* - \wstar_j) = (1{-}\rho)\wstar_j + \rho\wbar^*$.
\end{proof}

The concept-level risk becomes $\rho^2 B_j^2 + \sigma^2 dC/(Kn)$. Setting global $>$ concept-level: $(1{-}\rho^2)B_j^2 > \sigma^2 d(C{-}1)/(Kn)$, yielding the bound in Theorem~\ref{thm:noisy}. When $\rho \ge 1$ ($\eta \ge (C{-}1)/C$), concept-level is strictly dominated. \qed

\textbf{Proof of Corollary~\ref{cor:implicit}.}
Eq.~\eqref{eq:implicit_risk} follows directly from Lemma~\ref{lem:contamination}: the bias is $\|\widetilde{w}_j - \wstar_j\|^2 = \rho^2 B_j^2$ and the variance is $\sigma^2 dC/(Kn)(1{+}o(1))$ since the noisy groups contain approximately $K/C$ clients each.

The oracle concept-level risk is $R_{\mathrm{oracle}} = \sigma^2 dC/(Kn)$ (zero bias). The imperfect risk is $R_{\mathrm{imp}} = \rho^2 B_j^2 + \sigma^2 dC/(Kn) > R_{\mathrm{oracle}}$ always. The global risk is $R_{\mathrm{global}} = B_j^2 + \sigma^2 d/(Kn)$. The imperfect estimator beats global when $R_{\mathrm{global}} > R_{\mathrm{imp}}$, i.e., $(1{-}\rho^2)B_j^2 > \sigma^2 d(C{-}1)/(Kn)$---a \emph{relaxed} crossover (threshold inflated by $1/(1{-}\rho^2)$ relative to oracle). The bias penalty $\rho^2 B_j^2 = R_{\mathrm{imp}} - R_{\mathrm{oracle}}$ is fixed in $K$, while $R_{\mathrm{oracle}} = \sigma^2 dC/(Kn) \propto 1/K$. Thus $(R_{\mathrm{imp}} - R_{\mathrm{oracle}})/R_{\mathrm{oracle}} = \rho^2 B_j^2 Kn/(\sigma^2 dC) \to \infty$ as $K \to \infty$: the relative penalty grows, confirming the gap becomes the dominant cost. \qed

\textbf{Proof of Theorem~\ref{thm:proportional}.}
By random matrix theory, OLS with $N$ Gaussian samples and $d/N \to \gamma \in (0,1)$ has $\E[\|\what - \wstar\|^2] = \sigma^2\gamma/(1{-}\gamma)$. The two risks are $B_j^2 + \sigma^2\gamma_G/(1{-}\gamma_G)$ and $\sigma^2 C\gamma_G/(1{-}C\gamma_G)$. Algebra yields the bound in Theorem~\ref{thm:proportional}. \qed

\textbf{Proof of Theorem~\ref{thm:aniso}.}
In the spiked model, prediction risk variance decomposes into signal and noise subspaces. With concept separation in the signal subspace, the relevant variance is $\sigma^2 r s/N$ (not $\sigma^2 d/N$). The crossover balances $B_{j,\Sigma}^2 + \sigma^2 rs/(Kn)$ against $\sigma^2 rs/((K/C)n)$, yielding the bound in Theorem~\ref{thm:aniso}. \qed

\textbf{Proof of Theorem~\ref{thm:unified}.}
Combining contamination bias, anisotropic prediction risk, and proportional-regime variance: global risk is $B_{j,\pi,\Sigma}^2 + \sigma^2\gamma_G/(1{-}\gamma_G)$; noisy concept-level risk is $\rho^2 B_{j,\pi,\Sigma}^2 + \sigma^2\gamma_j/(1{-}\gamma_j)$. Setting global $>$ concept-level and simplifying yields~\eqref{eq:unified}. Recovery of the base case under $\pi_j{=}1/C$, $\rho{=}0$, $d_{\mathrm{eff}}{=}d$, $\gamma_j \to 0$ gives $\SNR > C{-}1$. \qed

\section{Proof of Proposition~\ref{prop:variance_inflation}}
\label{app:variance_inflation}

Consider a linear model $\what_j$ trained by $E$ epochs of full-batch gradient descent with step size $\alpha$ on $n_j$ samples from concept $j$ (or a contaminated mixture). The training dynamics decompose the excess risk as:
\begin{equation}
R(\what_j) = \underbrace{\|w_\infty - \wstar_j\|^2}_{\text{approximation bias}} + \underbrace{\sigma^2\mathrm{tr}(\Sigma(X^\top X)^{-1})}_{\text{irreducible variance}} + \underbrace{\Delta_{\mathrm{ovf}}(E, \alpha, n_j, d)}_{\text{optimization excess}},
\end{equation}
where $w_\infty$ is the population minimizer of the training loss. For the oracle, $w_\infty = \wstar_j$ (zero approximation bias). For the imperfect estimator, $w_\infty = (1{-}\rho)\wstar_j + \rho\wbar^*$ (Lemma~\ref{lem:contamination}), giving bias $\rho^2 B_j^2$.

\textbf{Overfitting penalty.} With $E$ epochs on $n_j$ i.i.d.\ samples, each gradient step moves toward the empirical minimum. When $n_j = O(d_{\mathrm{eff}})$, the model memorizes training noise, inflating generalization error. By standard bias-variance tradeoffs for early-stopped GD~\citep{ali2019continuous}, $\Delta_{\mathrm{ovf}} \propto \sigma^2 E \alpha d_{\mathrm{eff}} / n_j$ in the mildly overparameterized regime, giving $\partial\Delta_{\mathrm{ovf}}/\partial E > 0$ and $\partial\Delta_{\mathrm{ovf}}/\partial n_j < 0$.

\textbf{Regularization benefit of contamination.} The imperfect estimator's contamination mixes data from $C$ concepts. When the concept means differ, this contamination adds noise that is \emph{not correlated with concept-specific training noise}, effectively preventing the model from memorizing concept $j$'s specific noise patterns. This reduces the optimization excess by $\Delta_{\mathrm{reg}}(\rho) \ge 0$, analogous to label smoothing~\citep{szegedy2016rethinking} or dropout~\citep{srivastava2014dropout}.

\textbf{Crossover.} Setting $R_{\mathrm{imp}}^{\mathrm{SGD}} < R_{\mathrm{oracle}}^{\mathrm{SGD}}$:
$$\rho^2 B_j^2 + (\Delta_{\mathrm{ovf}} - \Delta_{\mathrm{reg}}) < \Delta_{\mathrm{ovf}} \implies \rho^2 B_j^2 < \Delta_{\mathrm{reg}}(\rho).$$
Since $\Delta_{\mathrm{ovf}} \propto 1/n_j$ and $n_j \propto K/C$:
\begin{itemize}[leftmargin=*,topsep=2pt,itemsep=1pt]
\item As $K/C \to \infty$: $\Delta_{\mathrm{ovf}} \to 0$, hence $\Delta_{\mathrm{reg}} \to 0$, and the bias $\rho^2 B_j^2$ dominates. The OLS result (oracle always wins) is recovered.
\item As $E$ increases: $\Delta_{\mathrm{ovf}}$ and $\Delta_{\mathrm{reg}}$ both grow, but $\Delta_{\mathrm{reg}}$ grows relative to $\rho^2 B_j^2$ (which is fixed), enlarging the regime where imperfect beats oracle.
\end{itemize}
We verify both predictions empirically: the CFL--Oracle gap shrinks monotonically with $K$ (\S\ref{sec:calibrated}) and grows with $E$ (Table~\ref{tab:epoch_sweep}, Appendix~\ref{app:epoch_sweep}). \qed

\section{Direct Shrinkage on Frozen Features}
\label{app:direct_shrinkage}

We validate the shrinkage estimator (Proposition~\ref{thm:shrinkage}) directly on CIFAR-100 frozen ResNet-18 features ($d{=}128$, $K{=}12$ clients, $T{=}20$ rounds), sweeping $C \in \{4, 6, 8\}$. Each round, we compute three estimators on linear heads: (i)~Global (pool all), (ii)~Concept-level (pool within true groups), (iii)~Shrinkage (Eq.~\ref{eq:shrinkage} with empirical $\hat\sigma_B^2$), plus a grid search over $\lambda \in [0,1]$.

At $C{=}4$: Shrinkage ($0.582 \pm 0.012$) $\approx$ Concept-level $\gg$ Global ($0.553 \pm 0.019$), with $\hat\lambda \approx 0.001$. At $C{=}8$ (near crossover): all three converge ($\approx 0.566$). The Concept--Global gap shrinks monotonically from $+2.9$pp ($C{=}4$) to $-0.1$pp ($C{=}8$), tracing the crossover. Grid search finds $\lambda^* \approx 0.3$--$0.6$, higher than $\hat\lambda$, consistent with the anisotropic setting where the isotropic formula is conservative.

\section{Oracle Intervention Decomposition}
\label{app:intervention}

A natural question is whether the gap between practical clustered FL methods and the oracle stems from \emph{incorrect concept assignment} or from other pipeline differences (model architecture, aggregation protocol, communication budget). We perform a causal decomposition on FedProTrack (FPT), a Bayesian concept-tracking method, by surgically replacing its inferred assignments with oracle labels.

\textbf{Setup.} CIFAR-100 disjoint labels, $K{=}10$, $C{=}4$, $T{=}30$, 5~seeds [42--46]. We test four FPT variants:
\begin{itemize}[leftmargin=*,topsep=2pt,itemsep=1pt]
    \item \textbf{FPT-base}: Standard FPT with Bayesian concept inference (spawns ${\sim}31$ concepts for 4 true).
    \item \textbf{FPT-OracleAssign}: Replace FPT's inferred assignments with true concept labels (concept models still maintained by the FPT pipeline).
    \item \textbf{FPT-CapSpawn}: Cap the maximum number of concepts at $C{=}4$.
    \item \textbf{Oracle}: True concept-level OLS (the theoretical benchmark).
\end{itemize}

\textbf{Decomposition.} The total gap Oracle $-$ FPT-base $= 5.7$pp decomposes as:
\begin{itemize}[leftmargin=*,topsep=2pt,itemsep=1pt]
    \item Effect of oracle assignment: $+0.07$pp (1.2\% of total gap)
    \item Effect of capping spawns: $+0.16$pp (2.8\%)
    \item Pipeline gap (model, aggregation, communication): $+5.63$pp (\textbf{98.8\%})
\end{itemize}

This decomposition was performed on FPT-\emph{base} (without calibration) in the disjoint-label setting---the most adversarial configuration for FPT, where it spawns ${\sim}31$ concepts for 4 true ones. In this regime, the pipeline gap dominates. However, in the shared-label setting that is the paper's main result (Table~\ref{tab:cifar_calibrated}), FPT-calibrated statistically matches Oracle ($0.471$ vs $0.472$, $p{=}0.84$), confirming that similarity calibration largely bridges the pipeline gap by providing adaptive shrinkage. The decomposition supports the paper's framing: the aggregation granularity question (Theorem~\ref{thm:crossover}) is logically prior to the concept-discovery question---a method must first determine \emph{whether} concept-level aggregation is beneficial before investing in concept inference.
